Remember Me Version 0.3
------------------------

Controls: 
	-A,S,D,W keys to move
	-Space Bar to dig chests
	-E key to open chest
	-Mouse pointer and click to open store or pause/resume game button.
	-Esc key to pause

This version of the game includes:
	-Playable character around the level island.
	-Collectible coins and Gems.
	-Chest Works
	-Usable Store.
	-Words count after being bought.
	-Pause Menu
	-Exit Game
	-Restart Demo
	-Sound Effects
	-Music
	-New Animations

Work in progress:
	-Progress Bar
	-End Level Condition
	-Options Menu
